# 小説『金魚』 構造分析レポート (Mckee / story-compiler 準拠)

## 📌 1. 基本構造と価値変化
- **主人公 (MC)**: ヒロセ（私）。精神疾患（障害）を抱え、薬を常用。他者との繋がりを求めている。
- **設定・舞台**: K大学、雪解けの季節（精神的な雪解け＝孤立からの離脱の象徴）。
- **価値変化の弧**: 
  - 【開始時】 孤立・神経症的（エアタバコ、薬、吐き気）
  - 【過程】 虚栄心と代替行為での接続（Discordの女性、喫煙所デビュー）
  - 【結末・到達点】 現実的な他者への能動的な参加（「一緒に勉強していいかな？」→「もちろん」）
  - **価値シフト**: 孤立（閉鎖） → 接続（開花）

## 📌 2. 統制概念 (Controlling Concept) とテーマ
- **統制概念 (テーマ)**: 人は欠落や狂気（病）を抱えながらも、ささやかな勇気と他者の受容によって、現実の世界（繋がり）へ回帰することができる。
- **対立概念**: 自身の病理や他者の狂気に引きずり込まれ、一生孤立と虚栄の中に閉じ込められる。

## 📌 3. キャラクターと機能（物語の三角構造）
McKee理論の「キャラクターは主人公の別の一面を引き出すために配置される」という原則に従い、主人公を取り巻く人物がきれいに配置されています。

1. **主人公（ヒロセ）**: 境界線上にいる存在。病の世界と正常な世界の間で足掻き、「喫煙」という儀式を通じて正常な世界へのパスポートを得ようとしている。
2. **引き摺り込む者 / 影（Discordの女性）**:
   - 主人公の「精神病・障害」の写し鏡。
   - 虚栄心を満たす道具であると同時に、自殺未遂・精神病棟という「あちら側」の引力を放つ。（＝「落ちてはいけない沼」の役割）
3. **日常と狂気の媒介者（拓也）**:
   - Discord経由の友人だが現実を共に生きる。
   - 自身も「自殺未遂を繰り返す青山にいる彼女」に振り回されているが、「可愛いからいいんだよ」と現実として受け入れてご飯（無料の白米）を平らげる。狂気を日常のカロリーに変換できている特異点。
4. **健全な日常の象徴（悠木・聡太）**:
   - キャンパスライフ、喫煙所での他愛ない会話、テスト勉強という「まともな社会」の象徴。
   - 主人公が最後に手を伸ばし、彼らもそれを受容する（「もちろん」）。

## 📌 4. 象徴 (Symbolism)
- **金魚**: 
  - 主人公の暗喩。「水槽（閉鎖空間）の中で、エアタバコのようにただ口をパクパクさせているだけの存在」からの脱却。
  - タイトル回収が秀逸。実際に煙草を吸い、現実に干渉することで「エアタバコ（金魚のパクパク）」が止まったという描写が、内面から外界へのステップを表している。
- **雪解け**: 冒頭の描写。主人公の硬直した人間関係や心性が溶け出すことのメタファー。
- **食事と吐き気**: 
  - 様々な匂い（他者の人生や混沌）に吐き気を催す（inゼリーしか飲めない）主人公が、気の置けない拓也とならラーメン（豚骨）を食べられる。他者への適応のバロメーター。

## 📌 5. 今後の予測（短編の続きがあるなら）
現在このテキストは、「孤立から小さな一歩を踏み出し、受け入れられた」という見事なビートで幕を閉じており、掌編として非常に完成度が高いです。
もしこれが長編の第1章（ACT1）である場合、以下の展開が予測されます。
- **インサイティング・インシデント（契機事件）の発生**:
  Discordの女性が現実の世界（K大学や主人公の生活圏）に侵食してくる、あるいは拓也の彼女の自殺が「嘘」ではなく現実になり、主人公たちの日常が脅かされる。
- **プログレッシブ・コンプリケーション（葛藤の増大）**:
  「喫煙所・勉強」という偽りの（あるいは背伸びした）健全さが通じない事態が起きる。主人公は再び薬と吐き気の世界に引き戻されそうになるが、悠木たちとの「繋がり」を武器に戦うか、手放すかの選択を迫られる。
