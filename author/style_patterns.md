# 文体・構造パターン集

> 著者: えあち / 分析日: 2026-03-09

---

## 1. 文体の5類型

### TYPE A：「内面叩きつけ型」独白（★ 最強の武器）

短文・体言止め・反復を積み重ねて感情を堆積させる文体。

**例（マチアプ）:**
```
誰のいちばんにもなれない自分
友達としては素敵だけど、恋愛対象としては見られない自分
想いが叶わず、たそがれている自分
そんな自分に酔っていた。
```

**構造:**
- 1〜2行の短文 × 3〜5連続
- 最後に「そんな自分に〇〇していた」で括る
- リズムが詩に近い → SNSでの引用・拡散に向く

**使用作品:** マチアプ, metamorphose, 大学までの人

---

### TYPE B：「妄想↔現実交差型」描写（★ 最も独創的）

主人公の妄想（暴力・加害・性的）が現実と交互に切り替わり、どちらが「現実」かが曖昧になる描写。

**例（metamorphose第1話）:**
```
脳の中で髪を引きずって座席の角に頭を打ちつける。
彼女はきっと、真っ青な顔をしながら、頭からあふれ出る血を必死に抑える。
〜〜（妄想）〜〜
そんな妄想をしながら歩く。
こんな気持ち悪い妄想をしている『化け物』なのに、
ご丁寧に、車内の人は僕がトイレに行くために、どいてくれる。（現実）
```

**構造:**
- 妄想パート（3〜10行）
- 現実に引き戻す1行（落差が大きいほど効果的）
- この繰り返しで「日常=狂気の隠れ蓑」を体現する

**使用作品:** metamorphose

---

### TYPE C：「詩的比喩・ファンタジー挿入型」（★ 短編向け）

主人公の感情を「頭の中のファンタジー」として物質化する手法。

**例（三角関数）:**
```
落書きで作った魚を、投げ飛ばした。
魚はすいすい泳いでいって、あんたのほっぺたに吸い付いた。
「ぷぷぷ」の文字列が風船になって教室の天井に滞留している。
```

**構造:**
- 現実の感情を「頭の中だけで起きていること」として描く
- それが相手に「触れる」（しかし相手は見えない）という孤独感の表現
- 「言葉が物になる」比喩 → 詩的

**使用作品:** 三角関数

---

### TYPE D：「社会格差・地域空気感型」（★ 長編向け）

地名・電車・方言・服装・内申点などのリアルな固有名詞で社会のテクスチャを作る。

**例（神戸を出る）:**
```
街並みは麓荘区とまるで違う。蔦だらけの古い公営団地、土汚れのついた、
お母さんが『ドカタ』と呼んでた作業員の人たち、ものすごい音で走り抜ける
暴走バイクの音、落書きだらけの高架。
```

**構造:**
- 視覚・聴覚・嗅覚を連続列挙
- 母親の言葉（偏見）を引用して「教育された差別感覚」を内面から描く
- 主人公が見る「外の世界」が、主人公の内面状態の鏡になっている

**使用作品:** 神戸を出る

---

### TYPE E：「視点転換・逆転型」（★ 連載の切り札）

同一の出来事を、別の人物の視点で再度描くことで意味が完全に反転する。

**例（metamorphose第3話）:**
- 第1・2話：黒屋（加害者）視点で「少女の優しさに崩壊する怪物」
- 第3話：少女視点で「お兄ちゃんを失った少女が、同じ目をした男を家に連れ帰った」
- → 黒屋が「救われかけた」のに「少女にもう一つのトラウマを作った」という転倒

**構造:**
- A視点：主人公の内面が中心、読者がある程度共感
- B視点：A視点での「善意・温かさ」が実は別の文脈を持っていたと判明
- ラストで両視点が「同じ踏切」に収束する

**使用作品:** metamorphose, マチアプ（第6話の北里エミリ視点）

## 2. ヒキの技法

### 冒頭フック（SNS最重要）

| パターン | 例 | 効果 |
|---------|-----|------|
| タブーの宣言 | 「なぜなら、簡単に振られることができるからだ。」 | 「は？」→読み進める |
| 定義の逆転 | 「僕にとって、恋愛とは被害者であることだ」 | 「どういう意味？」→続きを読む |
| 状況の謎 | 「僕は東京の下町に住んでいるのに、わざわざ福島まできて」 | 「なぜ？」→読み進める |
| 自己開示の過剰 | 「白鳥さんの脚をもぎ取ってしゃぶるんだよ！」 | 衝撃→読者の反応を引き出す |

### 話末フック（次話誘導）

マチアプの構造が特に優れている:
- 各話の末尾は「だから僕は彼女の透き通るような白い頬を思いっきり叩いた。」のような行動で終わる
- 「えっ、どうなる？」という未解決の緊張を残す

## 3. キャラクターの書き方パターン

### 主人公の「欠陥」の配置法

著者は主人公の欠陥を「社会が作った傷の副産物」として書くのが得意。

```
[傷のルート]
 子供時代の孤立・いじめ・過干渉
       ↓
 正常な感情回路の歪み
       ↓
 タブーな欲動（加害衝動・支配欲・自傷欲）
       ↓
 社会に溶け込むための「仮面」
       ↓
 仮面が崩れるきっかけ（ヒロインとの接触）
       ↓
 暴発・自壊
```

### ヒロインの「反転」パターン

著者のヒロインは必ず「主人公の予想」を外す：

| 作品 | 主人公の予想 | 実際のヒロイン |
|------|------------|-------------|
| マチアプ | 怒って振る | 「付き合ってみよう」と無表情で返す |
| metamorphose | 優しいだけの女の子 | 死んだ兄の代わりを求めて連れ帰った |
| 神戸を出る | 不良の怖い女 | 家族の借金と陸上の夢を背負った少女 |

## 4. 文章の長さ・リズムの設計

### SNSで受ける：短段落主義

- 1段落 = 1〜4行
- 改行を多用（縦スクロール読書に対応）
- 台詞と地の文を交互に短く
- 「沈黙」「余白」を一行の空行で表現

**良い例（マチアプ）:**
```
女の子「ご、ごめん、最近ちょっと予定が空いてなくて3月までわからないかも」

苦し紛れでそそる言い訳だ。

僕「じゃあ、3月以降会えるんだね！めっちゃ楽しみにしてるね！」

あえて満面の笑みで返事をする。
```

### 長編で失速するパターン

神戸を出るの中盤：
- 1段落が500〜1000字になる箇所が複数ある
- 心理描写と台詞が混在して追いにくい
- → 段落を50〜100字単位で強制分割すると読みやすくなる

## 5. note/SNS最適化のための文体チェックリスト

書き上げたら以下を確認：

- [ ] 冒頭1〜2行で「なぜ読むのか」がわかるか
- [ ] 1段落は4行以内か
- [ ] 台詞の前後に「行動・心理・空白」が挟まれているか
- [ ] 各話の末尾に「続きを読ませる緊張」があるか
- [ ] TYPE A（内面叩きつけ型）のリズムが一箇所以上あるか
- [ ] ヒロインは主人公の予想を外しているか
- [ ] グロ・性的描写は「テーマの補強」として機能しているか（目的化していないか）
