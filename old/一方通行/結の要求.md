### バッドエンドの基本方針

- [ ] あくまでバッドエンドとして終わらせたいです。
- [ ] 結なので余韻を残したいです。
- [ ] またテーマが「憧れは相手を理想化しているだけ」であるので、二人が「相手を理想化しすぎたのかなあ」と振り返るシーンは必須です。
- [ ] ただし、あくまでテーマなので、セリフとしてこの文言を明言はしないです。
- [ ] 二人の予後はとんでもなく悪いことを描いてください。
- [ ] この大箱では、破局から何年もあとを描いてください。

### 晃人の予後

- [ ] 晃人に関しては、兄の復讐という生きる理由を失ったので、ヤケクソになって、不貞腐れてアイドルもやめて生きていきます。
- [ ] 下手に完璧主義で、甘ったれなので、目的もなく普通に生きていくということができません。
- [ ] 資格の勉強を必死でして、なんとか見つけた際就職先でも、過去のゴシップを噂されて、嫌気がさしてやめてしまいます。
- [ ] 下手に完璧主義なので、居心地がいい環境でもないのにいるなんて自殺に向かって生きてるだけだと考えたからです。
- [ ] 甘ったれなので働かない自分を容赦します。
- [ ] それからは、甘ったれなので持ち前のルックスを生かして、甘ったれなので、ヒモ同然の生活をしたりするのですが、恋愛をするたびに茉莉とのトラウマを思い出し、相手を信じられず、相手から逃げ出してしまいお金も心を許せる相手もうしないます。
- [ ] そもそもヒモとして付き合うしかない時点で、ある程度相手に合わせなければならないし、相手も金を払ってるのだからと、本性を剥き出しにしてきがちです。
- [ ] そもそも甘ったれと完璧主義の相反する二つの性質を持つ彼は、ヒモとなって相手のご機嫌を伺って生きる生活やとか、ことすらままならない自分に葛藤します。
- [ ] そして何度も過去を振り返ります。
- [ ] あの時、茉莉を理想化しすぎてなければうまく付き合えたのかなとか、人と付き合ってなければこうはならなかったのかなとか、兄のことを信仰していなければとか、ありましないifルートに縋って酒を飲んで、真面目に生きれずどんどん社会の底辺に落ちていって、サラ金なんかしているところを匂わせて終わりです。

### 茉莉の予後

- [ ] 次は茉莉です。
- [ ] 風俗で散々イヤなおっさんたちに過去のことを掘り返されて、嫌気さして昼職をしてみようと試みるも、勉強も社会人スキルもなーんにもない茉莉はまともな就職先はありませんでした。
- [ ] 結局バイトに申し込むも、自分勝手でスキルも何もない茉莉はどこでもうまくいきません。
- [ ] それで誰とも会話しなくていい倉庫のバイトに流れ着いたものの、夜職時代と変わらない荒いお金の使い方をしてしまうので、結局過去のトラウマのある夜職にまた戻っています。
- [ ] 歳をとってストレスで老けて、持ち前の童顔も活かせなくなったのでかつてのような高級店には行けずさらに下品で汚いオッサンの相手ばかりします。
- [ ] 「白馬の王子様なんてどこにもおらず、底辺の私は汚いオッサンのちんぽを掃除するアバズレ女にしかならないのね」と自分を卑下します。
- [ ] そして年齢も30に近づき、婚活をしていくようになります。
- [ ] いい人が現れても，どうしてもルックスで好きになれません。
- [ ] 晃人の姿がよぎって高くなってしまった自分の理想に苦しみます。
- [ ] こうやって相手を理想化したり、理想を見すぎるから晃人の時も、今も何もかもうまくいってないんだ、と後悔します。
- [ ] あの時、晃人を振っていなければと、何度も何度も後悔します。
- [ ] 忘れようとしても、「あの人が晃人を振ったっていう茉莉さん？えーー、あんなに老けてるのにそんなわけないwww」なんて周りの嬢や、近隣の高級店の嬢に噂されます。
- [ ] 噂されて、でも今は私たちの格下だね、と馬鹿にされているのを聞いて惨めになります。
- [ ] 後悔に後悔を重ねて、晃人同様どこで間違ったんだろうと、毎日後悔しながらODやリストカット〈あくまで匂わせる〉を繰り返しながら生きていく様子を見せます。
