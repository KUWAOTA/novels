Constrain — ACT3 シーン描写例（20本）
時代：灘高校 → 東京大学
主人公：初めて「居場所」を見つける / 厳密さへの確信が固まる / Constrainの着想
A：連敗の蓄積 / 主人公への執着が変質する

凡例: 【場】=舞台・時間、【価値変化】=このシーンの価値の動き

================================================================================

S3-01「受験会場の朝」
【場】灘高校受験会場 / 冬の朝
【価値変化】使命感 → 試練の入口

試験会場は寒かった。受験番号を確認して席に着いた。隣の席の子がシャーペンを何本も並べていた。主人公は一本しか持っていない。壊れたら別の解法を考えればいい。問題用紙が配られた瞬間、他のことが全部消えた。B子のことも、A男のことも。問題だけが残った。これが一番楽だと思った。

================================================================================

S3-02「合格発表」
【場】インターネットの掲示板 / 自室
【価値変化】使命感 → 確信の強化

番号があった。主人公は三秒見て、ブラウザを閉じた。感情の爆発はなかった。ただ「仮説が正しかった」という静かな手応えがあった。父のノートを開いた。仮説を立てて、証拠を集めて、修正する。父が教えてくれた方法は正しい。だから次の仮説へ進む。

================================================================================

S3-03「A男の台所」
【場】A男の自宅 / 合格発表の夜
【価値変化】A男: 屈辱の蓄積

A男の合格発表は「不合格」だった。第二希望に変えるしかない。父親の声が低くなった時、A男は「あいつのせいだ」と思った。小学校からずっと。点数だけではない。父に愛される理由が、努力とは別のところにあるという理不尽。あいつが存在しなければ、自分は正しく評価されていたはずだ。——その論理に反論できなかった。

================================================================================

S3-04「灘高校の最初の朝礼」
【場】灘高校 / 入学式後の朝礼
【価値変化】孤立 → 居場所の発見の予感

「前にならえ」。全員が迷いなく動いた。でも今回は違う感じがした。隣の子が少しずれても、誰も笑わなかった。「うちの学校はそういうの気にしないから」と隣の子が小声で言った。「そういうの、とは？」「空気読むとか。だって結局頭いい方が勝ちじゃん」。主人公は少し黙った。そういう理屈で動く場所が存在するのか、と思った。初めて息がしやすかった。

================================================================================

S3-05「対等な会話」
【場】放課後の教室 / 数学の自習
【価値変化】孤立 → 真の繋がりの片鱗

「その証明、π使わずに書けない？」と後ろの席の子が聞いてきた。主人公は考えた。「……この前提を変えれば、できます」。「え、やってみて」。気づいたら二時間が経っていた。相手もずっと楽しそうにしていた。感情面での擦り合わせが一切なかった。ただ論理だけで会話が続いた。こういうことが、普通に起きる場所があるのか。

================================================================================

S3-06「厳密な言語の居場所」
【場】ディベート部の練習 / 放課後
【価値変化】使命感 → 「武器が武器じゃなくなる」感覚

部活の発表で、主人公は相手の定義の曖昧さを突いて勝った。拍手が来た。笑い声があった。でも嘲笑ではなかった。「お前の論理の切り口えぐいな」と先輩が言った。主人公がずっと「攻撃」と扱われていたものが、ここでは「技術」として評価された。初めて、自分の話し方が肯定された。B子の声が一瞬浮かんで、消えた。

================================================================================

S3-07「B子への後悔が返ってくる夜」
【場】自室 / 深夜
【価値変化】充足 → 奥底の喪失

昼間は楽しかった。でも夜になると、B子のことを考える時間がやってくる。あの子がここに来られたら良かったのに、と思う。ここなら大丈夫だったかもしれない。でもそれを証明する方法が、もうない。だから代わりに「厳密な言語のある社会を作る」という方向に変換する。変換しないと、ただ悲しいだけだから。

================================================================================

S3-08「Constrainの着想（原型）」
【場】図書館の個人ブース / 深夜
【価値変化】使命感 → 具体化の始まり

「言語の不備を正すツール」という構想をノートに書き始めた。単語の定義が揺れないシステム。曖昧な記述を自動的に検出するコンパイラ。最初は論文の形式を修正するツールとして考えていた。でも書いているうちに、もっと広い用途が見えてきた。これは、人が言葉で傷つけ合わないための道具になれるかもしれない。——Constrainという名前はその時生まれた。

================================================================================

S3-09「高校最後の夏」
【場】部室 / 夏休みの午後
【価値変化】充足の頂点 / ACT3のピーク

旧友と過ごす夏だった。正確には「友人」という言葉でいいのか今も分からないが、一緒にいて楽な人たちが何人かいた。三年間で初めて「楽しい」という感情が持続したような気がする。でも夏の終わりに、全員が受験に戻っていった。それが正しいことは分かる。

================================================================================

S3-10「東大受験の日」
【場】東京大学の受験会場 / 冬
【価値変化】使命感 → 集中の純化

前日は眠れなかった。B子の顔が浮かんで、父の顔が浮かんで、Constrainの設計図が浮かんだ。試験が始まった瞬間、全部消えた。そういう意味で、試験は好きだった。この瞬間だけは、全てが明確になる。

================================================================================

S3-11「A男の後期入試」
【場】東工大の受験会場 / 数週間後
【価値変化】A男: 屈辱の最大化

前期で東大を落ちた。後期で東工大に入った。親には「まあ仕方ない」と言われた。その「仕方ない」という言葉の重さを、A男は受験会場の椅子に座りながら考えた。あいつは受かったのだろうか。まだ確認していない。でも分かる。あいつは通った。いつもそうだ。なぜ。——なぜあいつは、努力しなくても正しい場所にいるのか。

================================================================================

S3-12「東大の入学式」
【場】安田講堂 / 春
【価値変化】使命感 → 確信の完成

来賓の挨拶を聞きながら、主人公は「この場所の言語水準は灘より高い」と感じた。誇張ではなく、会話の前提がより精密だということが肌で分かった。でも同時に「これでもまだ曖昧さが残る」とも感じた。だから自分がやる仕事はある。確信が静かに固まった。

================================================================================

S3-13「Constrainの開発開始」
【場】大学1年 / 自室 / 深夜
【価値変化】着想 → 具体的な構築の開始

Haskellで書き始めた。最初のバージョンは30行だった。全然動かなかった。でも面白かった。文章の中の曖昧さを検出する関数を書いていると、言語というものの構造が見えてくる気がした。これは正しい仕事だ。B子があの時「ごめんね」と言った意味が、そのとき初めて別の角度から見えた気がした——でも、その解釈はやはり間違っていた。

================================================================================

S3-14「初めての失敗と修正」
【場】研究室 / 学部2年
【価値変化】確信 → 一時的な挫折 → 修正能力の確認

教授に見せたデモが全く動かなかった。「定義の部分が甘い」と言われた。不快だったが正しかった。一週間で仕様を書き直した。「こっちの方がずっといい」と教授が言った。仮説を立て、検証し、修正する。父が教えてくれた方法は、コードでも動く。

================================================================================

S3-15「同期との差異」
【場】サークルの飲み会 / 大学2年
【価値変化】繋がり → 再び孤立の感覚

サークルに所属した。でも飲み会では会話の波長が合わなかった。「空気」を軸にした会話が、やはり苦手だった。冗談を「事実と異なる」と指摘した瞬間に場が白けた。天才と変人は隣り合わせだと誰かが後で言った。両方が本当かどうかは分からないが、どちらにしても少し疲れた。

================================================================================

S3-16「就職活動という選択」
【場】キャリアセンター / 大学4年
【価値変化】使命感 → 戦略的な選択

外資系のIT企業にしか興味がなかった。言語の精度が高い環境で働きたかったし、Constrainを実用化できるような基盤が欲しかった。Google Japanの説明会に行った。技術的な会話の水準が他と違った。ここだと思った。でも当然、競争は激しい。

================================================================================

S3-17「面接の手応え」
【場】Google Japan 面接 / 大学4年
【価値変化】戦略的確信 → 技術的自信

面接官が「この問題をどう解く？」と聞いた。主人公は仮説を三つ立てて、それぞれのトレードオフを説明した。面接官が少し乗り出した。「その分類は面白い、続けて」。——この人は論理を価値として見ている。ここに入れば、まともな会話ができる。

================================================================================

S3-18「Constrainのβ版完成」
【場】自室 / 卒業論文の前
【価値変化】使命感 → 達成感 / しかし本来の目的との乖離の始まり

Constrainの最初の動くバージョンが完成した。文章を入力すると、曖昧な表現・定義なし使用・論理矛盾を検出して警告を出す。テストケースとして自分の過去のレポートを入れたら、エラーが40個出た。笑えた。——でも同時に「これは人間の文章を裁くツールだ」という感覚が少し頭をよぎった。その感覚は、すぐに打ち消した。

================================================================================

S3-19「卒業式の日に」
【場】安田講堂 / 春
【価値変化】充足 → 次への準備

隣の席の子が泣いていた。主人公には涙は来なかった。感傷的である理由が見当たらなかった。これは通過点で、これからが本番だから。でも卒業証書を持ちながら、一瞬だけB子のことを考えた。あれから何年経ったのか。正確に数えた。——B子が死んだのがこれだけ前で、自分は今ここにいる。矛盾はないが、説明も難しい。

================================================================================

S3-20「A男の行動」
【場】A男の視点 / 大学卒業と就職の時期
【価値変化】A男の執着が次の段階へ

東工大を卒業したA男は、最初からリストを作っていた。あいつが就職しそうな企業。外資IT。GoogleかAmazonか。SNSに断片的な情報があった。A男はそれを辿った。尾行ではない、と自分に言い聞かせた。ただ確認したかっただけだ。あいつが自分より上の場所にいるのかどうか。——エントリーシートを出した。同じ会社に。

================================================================================
