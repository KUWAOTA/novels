# agents_story — AIエージェントへの注記

このフォルダに格納されているシーンファイル（ACT1〜ACT2）は、**AIが生成した草稿**である。

## 扱い方

- ユーザーが執筆・承認した正式原稿ではない
- 設定・描写・台詞は**参考資料として扱う**こと
- 「既に確定した描写」として引用・前提にしないこと
- 短編化タスク等で新規執筆する際は、これらに引きずられず、ユーザーの意図を優先すること
