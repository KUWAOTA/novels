Constrain — ACT2 シーン描写例（20本）
時代：中学時代
主人公：論理を武器に生き延びる / B子と出会い / 誤信念が確立される
A：いじめが激化 / 教育虐待も加速
B子：知的障害 / 低い自己肯定感 / 儚く優しい

凡例: 【場】=舞台・時間、【価値変化】=このシーンの価値の動き

================================================================================

S2-01「保健室の扉」
【場】中学1年 / 保健室の前の廊下 / 5月
【価値変化】孤立の継続 → 逃げ場の発見

教室に戻れない日が増えた。廊下を歩いていると、保健室の扉だけが少し開いていた。理由はない。ただ、開いていた。入っても怒られないのか考えながら、主人公はそこに立っていた。中からカーテンの擦れる音がした。誰かいる。でも声をかけるほど社会的な元気はなかった。その日は結局扉の外の壁にもたれて、授業が終わるのを待った。

================================================================================

S2-02「B子との初接触」
【場】保健室 / 昼休み
【価値変化】孤立 → 奇妙な接続の始まり

翌日、また保健室に来た。B子がベッドの端に座って、窓の外をじっと見ていた。「ここ、座っていい?」と聞いたら、B子は振り返らずに「どうぞ」と言った。二人は何も話さなかった。でもそれが苦痛じゃなかった。空気を読まなくていい相手というのが、こんなにも楽だとは思わなかった。

================================================================================

S2-03「B子の話し方」
【場】保健室 / 三日目
【価値変化】接続の深化 → 主人公による「鏡像」の誤認

「あなた、なんでここにいるの」とB子が聞いた。「教室が演算コストが高すぎて」と主人公が答えた。B子は少し考えてから「うん、私も難しい」と言った。B子の言葉はいつも少し遅くて、少しずれていた。でも主人公には、そのずれが奇妙に心地よかった。この子も自分と同じなのかもしれない、と思った。——それは誤解だった。

================================================================================

S2-04「A男の変化」
【場】廊下 / 昼
【価値変化】一時的回避の崩壊 → 危険の再燃

中学に入ってA男は変わっていた。父親に「主人公に負けるな」と言われて受験した私立が全滅だったせいで、怒りが別の質になっていた。廊下ですれ違う時の目が、小学校の頃より冷たかった。でも今はもっと巧妙だった。手は出さない。代わりに、言葉と関係を使う。

================================================================================

S2-05「噂が広がる」
【場】教室 / クラス替え後
【価値変化】中立 → 社会的排除

「あいつ、なんか怖くない？」「なんか自分の世界に入ってる感じ」「A男に聞いたら変な奴だって言ってた」。主人公の耳には入らないように計算されていた。でも全部聞こえていた。A男は暴力を評判に変えていた。そっちの方が効率がいいことに気づいたのだろう。なるほど、と主人公は思った。感情的にではなく。

================================================================================

S2-06「B子の虐め」
【場】校庭の隅 / 放課後
【価値変化】傍観 → 怒りの目撃

B子が別のグループの女子たちに何かを言われていた。聞こえなかったが、B子の顔から分かった。ゆっくり瞬きをして、何かを言われるたびに「ごめんなさい」と言っている。謝る理由がないのに謝っている。主人公はそれを見て、論理的に「不合理だ」と思った。でも足が動かなかった。関与する言語的準備がなかった。

================================================================================

S2-07「介入の失敗」
【場】翌日の廊下
【価値変化】正義感 → 加害の結果（意図せず）

「昨日B子さんに言っていたことは論理的に間違っている」と主人公は女子グループに言った。「彼女が謝る理由の根拠を示してほしい」。女子たちは黙って、それからB子を見た。「へえ、あんたって変な人の友達なんだ」。B子は小さくなった。主人公の介入が状況を悪化させた。言葉は正確だったのに、なぜ。

================================================================================

S2-08「二人だけの放課後」
【場】保健室 / 夕方
【価値変化】孤立の中の繋がり

放課後、保健の先生が出かけた後に二人だけになった。「なんで保健室に来てるの」とB子に聞いた。「決まりが分からないから」とB子は言った。「学校の決まりですか？」「全部の決まり。どこに立てばいいか、何を言えばいいか、全然分からない」。主人公は少し黙った。「それは俺も同じです」。その言葉を言った瞬間、何かが少しだけ解けた気がした。

================================================================================

S2-09「B子が受け取るもの」
【場】保健室 / 日常の一場面
【価値変化】B子: 受容の錯覚 → 読者には見える構造

「それは論理的に正しくない」と主人公がB子の言ったことを訂正した。B子はゆっくり頷いた。「そうか、ごめんね」と言った。主人公は「謝らなくていい、ただの情報の修正です」と言った。B子がふわっと笑った。主人公の言い方は世間では「きつい」とされるが、B子には「怒っていない人の声」に聞こえていた。それは彼女の自己像の問題だった。

================================================================================

S2-10「A男の親の印」
【場】A男の手首 / 更衣室 / 偶然の目撃
【価値変化】A男の苦しみの可視化 → 読者への揺さぶり

体育の着替えの時に、A男の腕に細い傷のラインを見たと思った。一瞬だった。A男がすぐに袖を下ろした。目が合った。A男は何も言わなかった。主人公も何も言わなかった。その日、主人公はノートに書いた。「A男の攻撃性には何らかの外因がある可能性。仮説：家庭環境」。感情ではなく仮説として記録した。だから何もできなかった。

================================================================================

S2-11「論理の盾」
【場】廊下 / いじめが始まる前
【価値変化】危険 → 一時的回避 / でも孤立が深まる

「なんで俺を避けるんだよ」とA男の取り巻きが言った。主人公は落ち着いて答えた。「避けているのではなく、接触のコストとメリットを計算した結果です」。相手が「は？」となる間に、主人公はもう角を曲がっていた。論理は使えば盾になる。でも、盾を使い続けると、守るべきものが中にあるかどうか分からなくなる。

================================================================================

S2-12「B子の虐めが激しくなる」
【場】廊下 / 昼休み
【価値変化】B子: 喪失の加速 / 主人公: 無力感

消しゴムが転がっていた。B子のだった。踏まれた跡がついていた。「ひどい」と主人公は言った。B子は「大丈夫だよ」と言った。「大丈夫じゃない。これは物理的な破損です」。「でも怒っても消せないし」。B子が笑った。その笑い方が、主人公には解析できなかった。泣いているのか笑っているのか、見分けがつかなかった。

================================================================================

S2-13「主人公がB子に勉強を教える」
【場】保健室 / 放課後
【価値変化】繋がりの深化 / しかし非対称の接続

数学を教えることになった。B子は理解が遅かった。主人公は何度も言い方を変えた。B子が少し分かった顔をするたびに——何かが主人公の胸の中で、論理とは関係ない動きをした。B子に分かってほしかった。そのことに気づいて、主人公は少し驚いた。誰かに理解してほしいと思ったのは、父以来だった。

================================================================================

S2-14「図書館のデート」
【場】近くの図書館 / 土曜日
【価値変化】繋がりの頂点 → 静かな幸福

B子と図書館に行った。「デート」という言葉は使わなかったが、今から考えるとそうだったと思う。B子は絵本のコーナーで長い時間立っていた。主人公は数学の本を読みながら、B子が笑っているのを時々確認した。帰り道、B子が「今日楽しかった」と言った。「俺もです」と答えた。言葉にするのに少し間が空いたが、それは嘘ではなかった。

================================================================================

S2-15「B子の言葉」
【場】帰り道 / 夕暮れ
【価値変化】幸福 → 静かな予告（主人公は気づかない）

橋の上で立ち止まって、B子が川を見ていた。「今日、死んでもよかったなあって思う」とB子が言った。主人公は一拍おいた。「それは比喩表現ですか？」「うん、多分」。「多分では困ります。生きることと死ぬことは物理的に不可逆なので」。B子がくすくす笑った。「あなたって面白いね」。主人公は笑い方が分からなかったが、悪い時間ではないと思った。

================================================================================

S2-16「翌朝のB子の空席」
【場】保健室 / 月曜日の朝
【価値変化】日常 → 不在の感知

月曜日、保健室にB子がいなかった。珍しいことではないかもしれないと思った。でも11時になっても来なかった。昼になっても。12時半に保健の先生が職員室に走るのが見えた。主人公はその走り方の意味が、すぐに分かった。「B子に何かあった」。分かってしまうのが怖かった。

================================================================================

S2-17「事実の確認」
【場】廊下 / 先生との会話
【価値変化】喪失の確定

「B子さんのことは聞きましたか」と先生が聞いた。「はい」と言った。「悲しいね」と先生が言った。「はい」と言った。悲しいという言葉が正確なのかどうか、その時には分からなかった。ただ、もう保健室に行っても意味がないということは、論理的に確定した。

================================================================================

S2-18「なぜの分析」
【場】自室 / 深夜
【価値変化】喪失 → 誤信念の確立

ノートに書いた。「なぜB子は死んだのか」。原因を分析した。虐め。理解されない日常。社会の曖昧さ。——「社会が十分に厳密であれば、B子は死ななかった」。もし、B子の状態に合わせた説明を誰かがしていたら。もし、言葉が正確に届く社会だったら。父のノートの言葉が頭の中で鳴った。「全ての誤解は言語の不備から生じる」。だから。だから自分がやるしかない。

================================================================================

S2-19「B子が幸せだったことを知らない」
【場】後日、B子の家族との会話（後日談）
【価値変化】誤信念の深化 / 読者には真実が見える

後日、B子の母親が言った言葉が耳に残った。「最後の週、すごく機嫌よかったんです。何か嬉しいことがあったみたい」。主人公はその言葉を「苦しみの中の虚偽の平穏」として解釈した。本当は苦しかったのを隠して笑っていたのだろう、と。——でも読者には分かる。B子は本当に、幸せだったのだ。

================================================================================

S2-20「中学を卒業する日」
【場】卒業式 / 体育館
【価値変化】喪失から使命感へ / 誤信念が完成する

卒業証書を受け取った。隣のクラスのA男が視界の端にいた。目は合わない方がいい。B子の席は空きのまま椅子が並んでいた。主人公はその椅子を見た。「厳密な言葉があれば、あの子は生きていた」。その確信はもう揺るがなかった。高校に入ったら、そしていつか、このために何かを作る。言葉を正確にするための道具を。
