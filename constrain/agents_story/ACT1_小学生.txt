Constrain — ACT1 シーン描写例（20本）
時代：小学生〜中学入学直後
主人公：ASD傾向あり・父の影響を受ける
A：努力型エリート・教育虐待を受ける

凡例: 【場】=舞台・時間、【価値変化】=このシーンの価値の動き

================================================================================

S1-01「朝礼の列」
【場】小学校 / 春の朝礼 / 年度初め
【価値変化】安全 → 孤立の予感

「前にならえ」の号令で子どもたちが自然と列に並ぶ。なぜそこに並ぶのかが主人公には分からない。誰かの後ろに立てばいいのか、どこに自分の場所があるのか——全員が迷いなく体を動かす中、主人公だけが一拍遅れて、慌てて誰かの後ろに割り込む。隣の子がちらりとこちらを見た。その目の意味が、分からない。

================================================================================

S1-02「比喩の誤解」
【場】小学校の教室 / 国語の授業
【価値変化】平静 → 孤立

「火が出るほど恥ずかしかった、という表現を使いましょうね」と先生が言った。主人公は正直に手を挙げた。「先生、火が出るって物理的に無理ですよね。体温は摂氏36度くらいなので」教室が静まりかえって、それからどっと笑い声が湧いた。主人公は何が笑いどころなのかが分からないまま、口を閉じた。

================================================================================

S1-03「『何でもいい』の罠」
【場】給食の配膳
【価値変化】参加 → 疎外

デザートはプリンかゼリーの二択で、係の子が「どっちでもいいよ」と言った。主人公は真剣に考えた。どちらでもいいなら、どうやって選ぶのか。結局ゼリーを選んだが、その後「何でもいいって言ったのに本当にどっちでもいいんかよ」と笑われた。どっちでもいい、とどちらを選んでもいいは違う。そこを笑われる理由が理解できない。

================================================================================

S1-04「リレーの失敗」
【場】運動会 / リレーの練習
【価値変化】期待 → 孤立

「バトンが来たら走り出しなさい」と言われた。主人公は正しくその通りにした——バトンが来た瞬間に走り始めた。しかし実際にはバトンが手に渡る少し前から走り始めないといけないらしく、チームは惨敗した。「お前のせいだ」。正しく言われた通りにしたのに。規則とは、言葉に書かれた通りではないのか。

================================================================================

S1-05「初めて嫉妬される」
【場】クラスの廊下 / テスト返却の日
【価値変化】無邪気 → 標的になる

主人公が100点のテスト用紙をそのまま机の上に置いていた。隠す理由が思い浮かばなかったから。A男がそれを見た。A男も96点だった。昨夜、父親に「なぜ100点が取れない」と言われて耳を引っ張られた。A男の目が細くなった。主人公はそれに気づかない。

================================================================================

S1-06「参観日の父」
【場】小学校の廊下 / 参観日
【価値変化】孤独 → 一瞬の安全

父が車いすを壁に沿って進ませながら教室に入ってきた。痩せていた。前回より少し痩せていた。でも笑っていた。主人公は黒板を指さして「今日また100点だったよ！」と叫んだ。父は笑いながら言った。「100点取れなくても、お前はいい子なんだよ」。廊下の端でA男がその会話を聞いていた。

================================================================================

S1-07「A男が聞いた言葉」
【場】参観日 / A男の視点
【価値変化】A男: 羨望 → 憎悪の種

父親から叱られ続けてきたA男には、あの言葉の意味が分からなかった。「100点取れなくても、いい子」。そんな理屈が存在するのか。自分はいつだって、点数がすべてだった。なのに、なぜあいつは——努力もせずに——あんな顔で父親に愛されているのか。理不尽だと思う前に、身体の奥でなにかが燃えた。

================================================================================

S1-08「父と数式」
【場】病院の面会室 / 月に一度の面会
【価値変化】孤立 → 繋がり（論理を通じた）

テーブルの上にノートが広げられた。父が鉛筆で小さな数式を書いた。「仮説を立てることが大事なんだ。まず自分なりの答えを決める。それから証拠を探す。違ったら修正する」。主人公はその言葉を書き写した。他の子が輪になって遊んでいる理由も、この方法で解明できるかもしれないと思った。

================================================================================

S1-09「最初の嫌がらせ」
【場】体育倉庫の影 / 放課後
【価値変化】安全 → 危険

A男と取り巻きの二人が倉庫の裏で待っていた。「なんでいつもお前だけ点数いいんだ」とA男が言った。主人公は考えた。「勉強しているからだと思います」。殴られた。痛いのは分かる。でも殴られた理由の論理が分からない。だから次も同じことを言うだろうし、きっとまた殴られるのだろうと思った。

================================================================================

S1-10「論理で返す」
【場】翌日の廊下
【価値変化】危険 → 一時的な回避 / しかし深まる対立

「お前を殴ったことを先生に言うよ」。主人公は静かに言った。「先生に言えばA男は呼び出しを受けて、親にも連絡がいく。A男の親は成績にうるさいから、呼び出しは都合が悪いはず。だからお前は今後殴らない方が合理的だよ」。A男の顔から血の気が引いた。効果はあった。でも、A男の目に宿った何かが、主人公には分からなかった。

================================================================================

S1-11「母の『努力』」
【場】夕食 / 台所
【価値変化】中立 → 主人公の誤信念の苗床

「テストで100点取れなかったのはもっと努力が足りないからよ」と母は言った。でも主人公は97点だった。3点は計算ミスで、計算ミスのパターンは既に分析済みで。「努力の方法が間違っているだけでしょ」と主人公は言った。母は言葉に詰まって、「……そうね、その通りかもね」と小さく言った。その瞬間、主人公は「意味不明な公理で動いている大人」というカテゴリを作った。

================================================================================

S1-12「A男の家の食卓」
【場】A男の家 / 夕食 / 主人公の知らない場所
【価値変化】A男: 緊張 → 暴力 → 恐怖

成績表が机の上に置かれた。父親がそれを見て、静かな声で言った。「なぜ一位じゃないんだ」。A男は「頑張りました」と言った。椅子が音をたてた。いつも静かにことが始まる。一位でなかったのは、あいつがいたから。あいつさえいなければ——。

================================================================================

S1-13「分析の成功」
【場】休み時間 / 教室の隅
【価値変化】孤立 → 偽りの充足感

ノートに書き溜めた仮説が当たり始めた。A男がいる時は視線を合わせない。あの子が近づいてきたら先に離席する。人間の動きにはパターンがある。パターンを解析すれば、地雷を避けられる。主人公は自分が賢くなった気がした。でも本当は、人との繋がりを全部回避しているだけだった。

================================================================================

S1-14「父の話す未来」
【場】病院の面会室 / 秋
【価値変化】孤立 → 一瞬の온기

「将来は何をしたいんだ」と父が聞いた。主人公は少し考えた。「言葉の精度を上げる仕事がしたい。みんなが同じ意味で言葉を使えるようにしたい」。父が目を細めた。笑ったのか、泣いたのかよく分からない表情だった。「それは難しい仕事だな」。「難しくても意味のあることが好きだから」。父が主人公の頭に手を置いた。

================================================================================

S1-15「最後の参観日」
【場】小学6年生 / 参観日
【価値変化】表面的な平穏 → 後から気づく別れ

この参観日が特別だと、その時は分からなかった。父は以前より疲れているように見えたが、それも「病気の人は疲れやすい」という知識の範疇だった。帰り際に父が言った。「来年も来るからな」。なぜ来年を強調するのか、その時は分からなかった。

================================================================================

S1-16「中学入学式」
【場】中学校の体育館 / 春
【価値変化】期待 → 疎外の継続

制服が窮屈だった。隣の席の子が「どうぞよろしく」と言ったので「こちらこそ」と返した。それだけで会話が終わった。笑顔をどこに向ければいいのか分からなかった。小学校より人数が多く、知らない顔ばかりだった。ただ、A男が別のクラスになったことだけは、論理と関係なく良かったと思った。

================================================================================

S1-17「父の訃報」
【場】中学校の廊下 / 初夏
【価値変化】日常 → 喪失

担任の先生が廊下で「一緒に来なさい」と言った。いつもと違う声のトーンだった。職員室に着く前に、もう分かっていた。「お父さんが——」。先生の言葉の続きは聞こえた。でも何かが脳の処理を止めたみたいに、ただ「そうですか」とだけ言った。先生が困惑した顔をした。泣かないのが変なのか、と後から思った。

================================================================================

S1-18「葬式で泣かない」
【場】葬儀場
【価値変化】喪失の中の麻痺

みんなが泣いていた。母も親戚も。泣くべき場面だということは分かっていた。でも涙が出なかった。悲しいのに。論理の回路が感情の出口を塞いでいる感じがした。「しっかりしてるわね」と誰かが言った。そうじゃない、と思ったが、うまく言えなかった。

================================================================================

S1-19「父のノートを開く」
【場】自室 / 深夜
【価値変化】喪失 → 誤信念の誕生

父の遺品の中にノートがあった。数式と仮説が書き連ねてあった。最後のページには「仮説：全ての誤解は言語の不備から生じる」と書いてあった。父が書いたものなのか、父が引用したものなのか、判断できない字だった。でも主人公はその一文を自分のノートに書き写した。これが正しい。誤解があるのは言葉が正確でないからだ。だから父も、正確な言葉があれば——。

================================================================================

S1-20「春の教室と空席」
【場】中学1年、父の死から一週間後の教室
【価値変化】喪失の確定 / ACT2への橋

朝、窓の外に桜が少し残っていた。誰かが「お父さん亡くなったんだって？」と後ろから声をかけた。主人公は「そうです」と答えた。「かわいそう」と言われた。何故かそれが一番きつかった。かわいそうという言葉は事実を変えない。父は死んだ。言葉の問題ではなく。でも言葉しか武器がない。だから、もっと正確な言葉を作らないといけない。

================================================================================
