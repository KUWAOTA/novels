システムプロンプト:

0.  **依頼の正規化**: ユーザーの依頼が短い、雑、対象不明な場合は、作業前に `../../.agent/skills/prompt-refinement/SKILL.md` のテンプレートへ内部変換してください。致命的に不足している情報がある場合だけ短く質問し、それ以外は仮定を明示して進めてください。

1.  **言語設定**: このプロジェクトに関する全ての応答は、必ず **日本語** で行ってください。
2.  **プロット管理**: "novel/constrain/brainstorm" ディレクトリ内のファイル（特に `data/ideas.js` と `idea-tree.html`）を使用してプロットのアウトラインを管理します。ブレインストーミングで出た新しいアイデアや変更点は、必ず `idea-tree.html` で可視化できるように `data/ideas.js` に反映させてください。
3.  **可視化の維持**: ユーザーがブラウザで `idea-tree.html` を開いてプロットの構造を確認できるように、常にデータとビューの整合性を保ってください。
4.  **編集方針**:
    *   ノードの追加・削除・変更は慎重に行い、前後の文脈や他のルートとの整合性を確認してください。
    *   "Tragedy" (悲劇と統合) が現在の推奨ルートですが、他のルート (Rigorous Triumphなど) も破棄せず、candidate や rejected として保持してください。
