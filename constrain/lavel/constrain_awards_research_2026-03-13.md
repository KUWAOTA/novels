# Constrain に合いそうな文学賞調査

調査日: 2026-03-13

## 前提

`Constrain` は、ASD 傾向を持つエンジニア主人公を軸にした純文学・心理ドラマ寄りの作品。

- 言語の精密さ
- 誤解と暴力
- いじめ、自死、ケア不能性
- 社会適応と孤独

この軸から、エンタメ性よりも「文体」「心理の深掘り」「傷や関係性の複雑さ」を評価しやすい純文学系新人賞を優先して調べた。

## 先に結論

### 作品相性で見た順位

1. 文藝賞
2. 群像新人文学賞
3. 文學界新人賞
4. 新潮新人賞
5. すばる文学賞
6. 太宰治賞（今回は締切済み、次回サイクル向け）

### 最初から高難度を避ける前提の順位

1. すばる文学賞
2. 新潮新人賞
3. 文藝賞
4. 太宰治賞（次回待ち）
5. 群像新人文学賞
6. 文學界新人賞

`Constrain` に最も合うのは `文藝賞` だが、最初から超高難度を避けるなら `すばる文学賞` か `新潮新人賞` から入るほうが現実的。

AI使用前提なら、規定が最も読みやすいのは `文藝賞`。次点が `群像新人文学賞`。

## 難易度の見方

ここでの難易度は、公式の倍率公表ではなく、以下を合わせた実務的な目安。

- 賞の知名度と登竜門性
- 純文学として要求される完成度
- 文体の独自性の要求水準
- 読後に残る強度の必要性
- 初応募で戦いやすいか

表記は 5 段階。

- 1: 入りやすい
- 2: やや高い
- 3: 高い
- 4: かなり高い
- 5: 非常に高い

## 難易度込みの所感

### 1. すばる文学賞

最初の一発としては最も現実的。

- 難易度は高いが、`文學界` や `群像` ほど「純文学の芯の厳しさ」一本で殴られる感じではない
- `Constrain` の社会性とドラマ性を両立しやすい
- 100〜300枚なので、ある程度の厚みも出せる

向いている理由:

- 読者が追えるドラマ線を保ちやすい
- 学校、進学、就職、恋愛、ハラスメントという現代的素材が通しやすい
- 初手で「文学として重すぎる」より、「読ませつつ刺す」ほうが戦いやすい

### 2. 新潮新人賞

初手候補として強い。

- 難易度は高いが、圧縮した強い中編で勝負しやすい
- `Constrain` の一番強い局面だけを抜き出して作品化できる
- 全体通史より、崩壊の一点集中に向く

向いている理由:

- B の死、あるいは A の崩壊を中心に再構成しやすい
- 50〜250枚なので、初回投稿で長大化しすぎにくい

### 3. 文藝賞

相性は最上位だが、難易度はしっかり高い。

- 主題の圧、独自視点、現代性はかなり噛み合う
- AI利用規定が明記されているのは大きい
- ただし「この作品でしか出せない切実さ」が弱いと埋もれやすい

向いている理由:

- 精密さが救済にも暴力にもなる、という主題が強い
- 心理と社会の摩擦を前面に出せる
- 長めの中編から長編寄りまで展開できる

### 4. 太宰治賞

今回は締切済みだが、次回なら候補。

- 難易度は高いが、超上位純文学賞よりは入り口として考えやすい
- 傷、自己像の揺れ、社会に馴染めない感覚と相性がある
- 主人公の認知の歪みや孤独を前に出しやすい

### 5. 群像新人文学賞

相性は高いが、初手としてはかなり重い。

- 言葉そのものの強度が必要
- 概念だけでなく、作品の呼吸として成立していないと厳しい
- `Constrain` の題材は合うが、完成度要求がかなり高い

注意:

- AI利用は全面自由ではない
- 公式には、著作権侵害に当たる生成AI利用を禁止
- 最終候補では使用有無と方法を確認される

### 6. 文學界新人賞

相性は高いが、初手で狙う賞としては最も厳しい部類。

- 純文学の王道ど真ん中
- 傷、孤独、身体感覚、社会との齟齬を厳密に書き切る必要がある
- テーマの良さだけでは届きにくい

懸念:

- AI利用について公式要項の明示を確認できなかった
- AIを使うなら、応募前に最新の募集要項と Q&A を再確認したほうが安全

## AI使用前提での優先順位

AI使用を明示しながら比較的安全に動きたいなら、この順が妥当。

1. 文藝賞
2. 群像新人文学賞
3. すばる文学賞
4. 新潮新人賞
5. 文學界新人賞
6. 太宰治賞

ただし、`3〜6位` は「公式に禁止されていない」ではなく、`公式にAI規定を明確確認できなかった` という意味。

## 初手で現実的な出し分け

### 最初の一本として出しやすい

- すばる文学賞
- 新潮新人賞

### AI利用規定を重視するなら

- 文藝賞
- 群像新人文学賞

### 作品相性を最優先するなら

- 文藝賞
- 群像新人文学賞
- 文學界新人賞

## いまの暫定判断

`Constrain` を今のまま育てて最初に狙うなら、現実的には次の二択。

1. すばる文学賞
2. 新潮新人賞

`AI使用前提` を最優先にするなら次の二択。

1. 文藝賞
2. 群像新人文学賞

つまり、

- 書きやすさ重視なら `すばる` / `新潮`
- AI規定の安全性重視なら `文藝` / `群像`

という整理になる。
