## 著者が描きたいこと（最重要）

この小説で実現したい描写の核：

1. **リアルな阻害の描写** — 主人公が世界と衝突する具体的な「うまくいかなさ」を、言い訳なくリアルに
2. **苦しみの描写** — 登場人物の苦しみを外から説明するのでなく、内側から体感させる
3. **論理や仮説で乗り越えていく** — 主人公の武器（厳密な言語・論理）を使って困難を突破する、クレバーで不器用な方法
4. **クレバーで不器用なリアリティのある描写** — 頭はいいが社会的にはうまくいかない、というリアリティを表面的でなく深く
5. **数多くの傷を負ってもなお世界の曖昧さを受け入れるという強い決断** — これが物語のゴール。「仕方なく受け入れる」ではなく「傷を知ったうえでの能動的な肯定」

## 小説「Constrain」の基本設定

### テーマ・価値弧
```
テーマ一文: 厳密さで世界を守ろうとした人間が、誤解と傷でできた社会を、それでも愛せるようになる話

価値変化弧: 理解すれば救える → 理解しても救えない → 理解できなくても寄り添える
```

### ACT構造
| ACT | 時代 | 主要出来事 |
|-----|------|-----------|
| ACT1 | 小学生 | ASD傾向で孤立・A男の嫉妬・父から論理を学ぶ・父の死（中学入学直後） |
| ACT2 | 中学時代 | B子と保健室で出会う・A男のいじめが激化・B子の自殺・誤信念確立 |
| ACT3 | 灘高校〜東大 | エリートコミュニティで適応・B子を救えたという後悔・厳密さへの確信 |
| ACT4 | 社会人 | Google Japan・Constrain開発・A男（同期→部下）への善意の教育→崩壊 |
| ACT5 | クライマックス | A男が自殺企図・主人公が論理を手放して寄り添う・エンディング（面会室） |

### 登場人物

**主人公**
- ASD傾向（作中では明示しない）
- 信念: 「正確に理解すれば人は救える」「誤解 = 言語の不備」
- 武器: 言語の厳密さ、論理、仮説
- 父（京大出身・研究者・ALS・中1で死去）が唯一の理解者
- 灘高→東大→Google Japan エンジニア
- 「Constrain」（Haskell製の文章コンパイラ）を開発・運用

**A男（いじめっ子→Google同期→部下）**
- 教育虐待（学年1位でないと暴力）を受けた
- 両親の夫婦喧嘩をうかがって育ち → 空気読みが異常に発達
- 主人公への嫉妬: 努力せず成績が良い + 父に愛されている
- 東工大（東大落ち）→ Google Japan（コミュ力採用）
- 主人公が「善意の教育」をすることで父親の教育虐待を追体験し崩壊
- 崩壊の5段階: 睡眠 → 仕事 → 親 → 金 → 自己像
- ACT5で自殺企図

**B子（中学時代の恋人・故人）**
- 知的障害（感情と表出がズレる：嬉しくて泣く・悲しいのに笑う）
- 自己肯定感が極めて低く、主人公の直截な物言いを「優しさ」と受け取る
- 主人公との最高に幸せなデートの日に、「幸せの絶頂で死にたい」と思い自殺
- 主人公は「社会の厳密さのなさが彼女を殺した」と誤解する（これが誤信念の根拠）

**父（故人）**
- ALS（筋萎縮性側索硬化症）で中学入学直後に死去
- 主人公に論理・仮説・数式の美しさを教えた
- 「100点取れなくても、◯◯はいい子なんだよ」→ A男の嫉妬の火種
- 京大出身の研究者

**母**
- 「努力」信奉者
- 主人公に「頑張れ」を押しつける
- 主人公がA男にすることと同じ構造のことを主人公にしていた（鏡像）

## アイデアの書き方フォーマット

```
001. [タイトル（10字以内）]：[説明（1〜3文で具体的に）]
```

例：
```
042. 数学の問題で安心：主人公は感情的に混乱した時、数学の問題集を解く。答えが一つに定まる世界に入ることで、曖昧さで満ちた現実から一時的に撤退できるからだ。机の引き出しに常に問題集が入っている。
```

## 実現したいトーンの参考

著者の過去作「metamorphose」「マチアプ」（old/ディレクトリに格納）で実証されたトーン：
- タブーな内面を外側から説明せず、一人称で「そのまま」書く
- 社会の「普通」に傷つけられた人間の歪みをリアルに
- 「嫌悪感」と「共感」を同時に生む二重構造
- 予想を裏切る展開や視点転換

Constrainでこれを純文学レベルに引き上げる：
- 主人公の「論理で生きる人間」の内面を、論理の言語で書く（内側から）
- 「正しいのに間違っている」構造をシーンで体験させる
- 言語 vs 身体、理解 vs 抱擁の対比を具体的なシーンに落とし込む
