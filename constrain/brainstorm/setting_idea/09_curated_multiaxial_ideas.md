# 厳選・多軸アイデア集

001. [父の代筆]：ALS が進んだ父が、自分で書けなくなった研究メモを主人公に口述筆記させる。主人公は「正確に転記すること」が愛情だと学び、その経験が後年の Constrain 開発思想に直結する。補強理由：愛情と厳密さが最初は一致していたと示せるため、後半でそれがズレた時の悲劇性が増す。追加意図：主人公の誤信念を単なる性格ではなく、愛された記憶から出たものとして採用判断できるようにする。根拠：Story Compiler `06_protagonist` の mentality linkage、`05_controlling-concept`、McKee『Story』の価値起源。

046. [主人公の遅い気づき]：A男の異変に主人公が気づくきっかけは、成果低下ではなく「質問が減ること」。理解しようとしなくなった時、人は壊れていると知る。補強理由：主人公らしい観測点で変化を捉えられる。追加意図：クライマックスへの因果を主人公の認知に沿って作る。根拠：Story Compiler `14_causal-chain`、`19_scene-mastery`。

047. [保健室と面会室]：保健室では B子 と並んで座れたが、面会室では A男 と向かい合うしかない、という構図の違いを使う。最後に主人公が椅子を少しずらし、真正面をやめる。補強理由：空間配置だけで「理解から同席へ」の転換を見せられる。追加意図：テーマを構図で採用判断できるようにする。根拠：Story Compiler `17_symbolism-and-image`、`19_scene-mastery`。

048. [終幕の未解決]：ラストで A男 が主人公を完全には許さず、主人公も自分を完全には赦せないまま終わる。ただし同じ部屋にいられる。補強理由：世界の曖昧さを能動的に受け入れる結末に合う。追加意図：「理解すれば救える」へ戻らないための歯止めにする。根拠：Story Compiler `05_controlling-concept`、McKee『Story』の anti-didactic ending。
