# 主人公設定の査定ガイド

`01_protagonist_settings.md` は件数が多いため、次の順で分割整理する。

## 優先して残す案

- 主人公の阻害が身体感覚で出る案
- 父から受け継いだ論理が愛情でもあったと分かる案
- B子には救いとして機能し、A男には暴力として返る案
- ACT5 の「未理解を欠陥扱いしない」転換を早期に伏線化できる案
- Constrain 開発やレビュー文化に直結する案

## 優先して削る案

- 外見や癖だけで止まる案
- ASD 傾向のラベル説明に近い案
- 他キャラとの関係変化を生まない案
- 既存案の言い換えで、圧力下の露出が増えない案

## 査定時の見方

各案を次の形式に直せないなら、削除候補にする。

```md
001. [タイトル]：内容。即する軸: 1,3,4。SC: `06_protagonist`, `21_character-dimension`。効用: 主人公のどの誤信念や転換伏線に使うか。
```

## 先に拾うべき章別機能

- ACT1: 阻害の原型、父との愛着、A男からの視線
- ACT2: B子には届く成功体験、制度への介入失敗
- ACT3: 厳密さが成功として強化される誤学習
- ACT4: 論理が善意のまま暴力化する実務場面
- ACT5: 論理の放棄ではなく、限界認識へ向かう伏線
