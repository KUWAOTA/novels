## 1. 物語の核心テーマ（変遷）

### v1 — 初期テーマ（idea-tree初期）
```
「再現性と曖昧さ」の対立
ASD的思考の主人公が、曖昧さにも価値があると気づく話
```

### v2 — 歩き方のメタファー（value_map v3）
```
「歩き方は一つではない」
・主人公の足 = 義足（精度・言語）
・いじめっ子の足 = 素足（空気を読む力）
・主人公の過ち：「義足の方が優れている」と信じ他人の素足を切って義足を強制した
・着地点：歩き方は足でも義足でもいい。他人の足を切らなくてもいい。
```

### v3 — 現在採用（girlfriend.md 最終版）
```
「厳密な理解を必要とする人間が、不完全な社会と共存する方法を見つける話」

価値変化弧：
  理解すれば救える
  ↓
  理解しても救えない
  ↓
  理解できなくても寄り添える

テーマ一文：
  厳密さで世界を守ろうとした人間が
  誤解と傷でできた社会を
  それでも愛せるようになる話

キーワード：
  絶望を理解した後の肯定（希望の物語ではない）
  制御 → 共存
  言語 → 身体（クライマックス）
```

## 4. クライマックス候補と変遷

### 現在の採用クライマックス（girlfriend.md最終）

**「撃たれながらも抱きしめる」**

```
行動の制約（この行動が成立する6条件）：
①厳密さを否定しない
②恋人の死の意味を保つ（厳密さは必要だった）
③いじめっ子の苦しみと向き合う
④状況を正確に理解している（衝動ではない）
⑤人間社会を拒絶しない
⑥社会全体ではなく目の前の人に行動する

価値変化：
  合理 → 非合理
  理解 → 愛
  言語 → 身体

なぜこれが強いか：
  主人公の哲学が身体で裏返る瞬間
  「論理の極致 → 論理の限界 → 身体の選択」
  キリスト的自己犠牲に見えないための前準備が必要
```

**クライマックスを成立させるための5つの準備：**
1. いじめっ子の崩壊の予兆（段階的に描く）
2. 主人公の価値観の揺れ（論理では救えなかった小失敗）
3. 主人公といじめっ子の鏡像関係の確認（「この人は俺に似ている」）
4. 抱きしめる行為の縮小版の予兆（途中のシーンで）
5. 社会の小さな善意の断片（人間社会を肯定する材料）

## 5. 構造分析（McKee）

### 三角構造

```
恋人（鏡像）
  役割：厳密な理解があれば生きられた人
  主張：世界は厳密でなければならない
  三角での機能：厳密さは「救い」の証拠

主人公
  役割：厳密の担い手
  世界モデル：正確に理解すれば世界は救える
  クライマックスでの回答：？？？ → 「それでも愛する」

いじめっ子（逆像）
  役割：厳密の世界で壊された人
  主張：世界はそんなに整理できない
  三角での機能：厳密さは「暴力」の証拠
```

**なぜ三角構造が強いか：** 二人だと「正しい/間違い」になる。三人だと「三つとも正しい」状況が作れる → 倫理的な深さを生む。

---

### 主人公の誤信念（Misbelief）

```
初期世界モデル：
  身体で世界を進めない
  ↓
  言語で世界を制御する
  ↓
  言語を厳密にすれば誤解はなくなる
  ↓
  誤解がなければ人は傷つかない

隠れた前提（後に崩れる）：
  誤解 = 言語の不備（実際は：誤解 = 人間の条件）

崩れ方：
  厳密に理解しようとする
  ↓ その行為が人を追い詰める
  ↓ 取り返しのつかない結果（クライマックス）
  ↓ 「理解できない」が初めて受け入れられる
```

---

```
社会が厳密なら救える（恋人の死 → 誤信念の確立）

厳密で社会を変えようとする（Constrain開発）
↓
厳密さが別の傷を生む（いじめっ子への善意の暴力）
↓

それでも人間を愛せる（「抱きしめる」という選択）
```

### マッキー的クライマックム条件チェック

**旧クライマックス「マスクを外して叫ぶ」：**
- ✓ 主人公のアクション（能動的選択）
- ✓ 不可逆性（見られたら取り消せない）
- ✓ 最大プレッシャー
- ✓ 実像の暴露
- ✗ 価値変化軸のズレ（抑圧→解放 になってしまいテーマと不一致）

**新クライマックス「撃たれながら抱きしめる」：**
- ✓ 主人公のアクション（逃げずに留まる能動的選択）
- ✓ 不可逆性（撃たれた / 抱きしめた事実）
- ✓ 最大プレッシャー（死ぬかどうか）
- ✓ 哲学が身体で裏返る（論理家が論理ではなく体で行動）
- ✓ テーマ一致（合理→非合理、理解→愛、言語→身体）
- △ 前準備なしだと「都合がよい」に見えるリスク（5つの準備が必要）
