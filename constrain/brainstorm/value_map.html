仮プロット + 価値要素マップ v3

中心テーマ ── 歩き方のメタファー

🦿 世界の歩き方は一つではない

主人公の足 ＝ 義足（精度）

ASDで「普通の足」（共感・空気読み）が使えない。
言語・数式・コードという義足で歩く。
世界を高い解像度で見られる。

≠

いじめっ子の足 ＝ 素足（空気読み）

虐待家庭で「空気を読む力」を生存本能として獲得。
低IQで義足は履けない。でもそれが彼/彼女の歩き方。

主人公の過ち：「義足の方が優れている」と信じ、他人の素足を切って義足を強制したこと。善意で。
着地点：歩き方は足でも義足でもいい。他人の足を切らなくてもいい——困っている人がいれば、その人に適した歩き方を教えればいい。
敵など誰もいない。攻撃する人間には満たされなさがあるだけ。
キャラクターと価値の配置
主人公（CTO）
ASD 言語の精度 マスク強迫症 善意の加害者
自分の感情が読まれることへの恐怖から強迫的にマスクを外せない。
「義足（精度）こそ唯一の正しい歩き方」と信じ、いじめっ子を善意で追い詰める。
クライマックスで生まれて初めてマスクを外す。
いじめっ子
虐待被害者 空気読みの達人 低IQ・庶務派遣 親を殺害
崩れた家庭で空気読みを生存本能として獲得。低IQで高い教育を受けられず。
虐待した親を介護するためリモート庶務として主人公の会社に。
善意の教育に対抗できずうつ病→依存する親を殺害→自殺へ。
父（故人）
人格者・救世主 愛の源泉 中高時代に死去
主人公を無条件に愛した唯一の存在。主人公の言語獲得を支えた。
いじめっ子を悪く言うと「悲しそうな目」をした——言語ではなく表情で。
主人公はその意味を、クライマックスまで解読できない。
母
曖昧の象徴 善意 情緒不安定
悪意なく主人公を守ろうとするが、感情的で振り回す。善意のおせっかいで怒る。
主人公の「曖昧さ＝悪」の確信を強化する存在。
※主人公はいじめっ子に、母と全く同じことをしている。
仮プロット ── タイムライン
第一章 ── 幼少期
いじめの始まり
いじめっ子は主人公の「空気の読めなさ」を武器に攻撃する。
動機は主人公と父との愛の関係への嫉妬。「あいつは僕より劣っているのに愛されている」。
母は感情的に守ろうとするが、情緒的に振り回す。
父はいじめっ子を悪く言う主人公に、言葉ではなく悲しそうな目で返す。
安全 → 危険 愛される → 攻撃される 父の目 = 未解読メッセージ
第二章 ── 少年期
「義足」の獲得
世界を精密に記述する言語・数式・コードを覚える。義足で歩けるようになる。
父がその成長を見守り、支える。これが主人公にとって最も安全な時間。
しかし「曖昧さ＝悪」という確信も同時に育ってゆく。
感情が読まれることへの恐怖 → 強迫的にマスクが外せなくなる。
無力 → 力を得る 理解 → 支配の萌芽
第三章 ── 中高時代
父の死
唯一の安全基地が失われる。
父の「悲しそうな目」の意味は未解読のまま封印される。
「曖昧さ＝悪」の確信は修正される機会を失い、固定される。
主人公は独力でもがかなければならない。
愛される → 喪失 導かれる → 独力で歩む
第四章 ── 青年期（20代）
いじめっ子の再登場 ── 支配構造の逆転
主人公はCTOとしてベンチャーを率いる。言語（コード）で世界を高解像度で見ている。
いじめっ子はリモート庶務の派遣社員として部下になる。田舎の親を介護するため。
主人公は善意から「教育」を開始。厳密に答えられるまで離さない。
いじめっ子は主人公が善意だとわかる（空気読みの達人）——だから対抗できない。
母が主人公にしたことと、主人公がいじめっ子にしていることは同じ構造。
被害者 → 加害者（無自覚） 善意 → 善意の暴力 主人公 ＝ 母の鏡像
第四章 後半
いじめっ子の過去を知る
いじめっ子が虐待被害者であったことを主人公は初めて知る。
空気読みは生存本能だったこと。低IQで「厳密に」は本質的に難しいこと。
主人公は自分がかつてされたことと同じことをしていたと認識し始める。
しかし「厳密な言動でもいじめっ子を救えると思ってしまう」——まだ気づけていない。
見えていなかった → 見え始める 父の目の意味が迫る
第五章 ── 危機（Crisis）
いじめっ子の崩壊——両親殺害
善意の教育によりうつ病発症。稼ぐ自信を完全に喪失。
虐待した親を介護しながら、追い詰められていく。
ついに依存してきた両親を殺害する。（依存の果ての暴力 = 切り離せなかった繋がりの断絶）
血のついたナイフを持って、主人公の会社のビルへ来る。
依存 → 殺害（究極の断絶） 善意の教育が引き金を引いた
第五章 ── クライマックス（CLIMAX）
🔥 マスクを外す——「お前に死んでほしくない」
ビルのロビーor廊下。血だらけのナイフを持ったいじめっ子が現れる。
周囲は逃げる。主人公だけが残る——背景を知っているから。

「お前に悪意があったわけじゃないって、わかってた。
ずっと羨ましかった。
……殺せないよ。知ってたよ。じゃあね。」
泣きながら言って、屋上へ向かう。

主人公、追いかける。
論理で止めようとする。言葉を組み立てようとする。
——構築できない。初めて、精度が届かない場所に立つ。

「お前に死んでほしくないんだあああああ——！」
マスクを外して、叫ぶ。
義足を脱いだ。防衛を捨てた。見られてしまった。取り返しがつかない。

いじめっ子、止まる。
空気読みの達人として、その叫びの中に「本物」を感じた。
誰かが自分のために、できないことをしている——初めて。
精度 → 叫び マスクを外す = 義足を脱ぐ 父の目の意味が今わかる 主人公の加害の結果を目の前に見る
エンディング ── 数年後
刑務所の面会室——抱擁
いじめっ子は逮捕・収監。
数年後、主人公が面会に来る。（理屈ではなく体で動く選択）

世界の解像度が上がり、世界が鮮やかに見える。
義足でも素足でも歩ける。他人の足を切らなくてもいい。
敵など誰もいなかった。

ガラス越しの面会室で——抱擁。
共に歩む 理解 → 本当の理解 父の悲しそうな目の意味、完全に解読
価値要素の対立軸
中心軸
共に歩む ⟷ 自分の歩き方を強制する
精度 vs 曖昧ではない。
「異なる認知のあり方を認め合えるか」が本質。
行為の本質
教育（適した歩き方） ⟷ 善意の暴力
「あなたのために言っている」が最も残酷な暴力。
悪意なら対抗できる。善意には対抗できない。
根源的欲求
愛される ⟷ 愛されない
主人公＝父に愛された → いじめっ子の嫉妬の原点
いじめっ子＝愛されなかった → 虐待した親にすら依存
幼少期の鏡像
「空気を読め」 → 読めない（ASD）
いじめっ子が主人公にできないことを強いた暴力
青年期の鏡像
「厳密に答えろ」 → 答えられない（低IQ）
主人公がいじめっ子にできないことを強いた暴力
= 全く同じ構造
依存と自立
自立・自由 ⟷ 依存（離れられない）
いじめっ子は死んでも親から離れられなかった。
殺害すら「依存の断絶」の最終形。
否定の否定（McKee）
ポジティブ 教育・導き → その反対 放置・無関心 → 否定の否定 善意の教育で人を殺す
ポジティブ 理解 → その反対 無知 → 否定の否定 「理解」という名の支配
ポジティブ 共に歩む → その反対 孤立 → 否定の否定 他人の足を切って自分の足で歩かせる
ポジティブ 愛される → その反対 愛されない → 否定の否定 愛された者が、愛されなかった者を死に追い込む

2

3
「殺せないよ知ってたよ」の意味を深める。主人公を殺せないのか、自分を殺せないと思っていたのに殺せたのか、どちらを指すのか。
4
エンディングの時間軸。数年後の刑務所面会室——主人公は何を経て面会に来るのか。その間の主人公はどう変化するのか。
5
時代設定の確定。97年生まれと同じか少し前か。いじめ・発達障害への不理解を描くために時代を遡らせるか。
