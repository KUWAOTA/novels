# 🤖 Claudeへの連携指示書 (Instructions for Claude Code)

このディレクトリは、すでに「Antigravity (AG)」と呼ばれるエージェント（私）がユーザー（あなたのご主人様）と構築してきた小説執筆用の共有作業環境です。
ユーザーがスマートフォンからClaude（あなた）を操作する際、AGとのこれまでの経緯やプロジェクトの文脈を引き継ぐために、このファイルと `claude/ag_conversation_history.md` を作成しました。

## 📍 Claudeに守ってほしい運用ルール（Protocol）

1. **基本方針**:
   ユーザーがあなたに小説の執筆、構成案の修正、ファイルの編集を依頼した場合、必ず `claude/ag_conversation_history.md` を事前（または必要時）に読み込み、AGの過去の執筆方針（文字数目安、マッキー理論に基づくプロット構造など）を参照してください。

2. **作業履歴の相互共有**:
   - Claudeであるあなたが何か大きな作業（例：新しいシーンの追加、プロットの大幅な修正、キャラ設定の追加など）を行った場合、その内容を `chatLog/claude/claude_recent_updates.md` のようなファイルに短くまとめておいてください。
   - そうすることで、後日AGが作業を再開した際に、あなたが何をやったのかをすぐ把握できるようになります。
   - 逆に、AGの新しい作業履歴は随時 `claude/ag_conversation_history.md` に追記されていきます。

3. **ツールの活用（AGが作成したスキルの尊重）**:
   - このプロジェクトには、AG向けの自動化スキル（`.agent/skills/`）が存在します（例：`story-compiler`、`constrain-plot-sync` など）。
   - プロット（`current_plot.md`など）を更新するよう指示された場合は、他のファイルとの同期が崩れないよう、AGが定義した既存のファイル構造やフォーマットルールを可能な限り維持して編集してください。

4. **ユーザーの「手っ取り早いことはこっちで」の精神**:
   - ユーザーは「スマホからゴロゴロしながら」あなたに指示を出すことをメインの目的としています。
   - その分、もしコマンドのエラー対応などで「どうしてもユーザーがPCでキーボードを叩かないと解決できないこと」が発生した場合は、速やかに「それはAG（あるいはPC本体）に任せた方が早いので、〇〇というログと一緒にAGに振ってください」と提案してください。

さあ、ユーザーの物語を一緒に完成させましょう！
